# mifos-mobile-app

I've enhanced the mifos mobile app by adding features such as toggle button, haptic feedback and dynamic theming which was not present earlier in the previous mobile app. Mifos mobile app is used globally by financial institutions and schools.

I've developed these features locally only - experimenting with it and haven't pushed in mifos mobile repo.
